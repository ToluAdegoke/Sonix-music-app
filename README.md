# 🎵 Sonix — Flutter Music Player App

A Flutter-based music player application built as a personal learning project to explore mobile development, API integration, and UI design.

## 📱 Overview

Sonix is a cross-platform music player built with Flutter and Dart. It focuses on delivering a clean, modern interface for browsing and playing music using data from the Deezer Public API.

The project was developed to strengthen my skills in Flutter development, state management, and working with real-world APIs.

## ✨ Features

- Browse trending tracks, albums, and artists
- Search for music content in real time
- Music playback using 30-second previews (Deezer API)
- Full-screen music player with album artwork
- Mini player for quick playback control
- Library for liked songs and playlists (stored locally)

## 🧰 Tech Stack

- Flutter & Dart
- Provider (state management)
- just_audio (audio playback)
- HTTP (REST API integration)
- SharedPreferences (local storage)
- Cached Network Image

## 🌐 Data Source

This app uses the Deezer Public API to fetch music data and preview audio clips. It supports:

- Track, album, and artist search
- Charts and curated playlists
- 30-second audio previews for playback

## 🧠 What I Learned

- Building cross-platform mobile apps with Flutter
- Consuming and handling REST APIs
- Managing state in medium-sized applications
- Designing reusable UI components
- Handling audio playback in mobile apps
- Structuring a scalable Flutter project

## 🚧 Project Status

This project is partially completed. Core features such as UI, search, and audio playback are implemented. Advanced features like background playback, offline downloads, and enhanced playlist management are planned for future development.

## 🐛 Limitations

- Only 30-second previews are available due to API restrictions
- Some tracks may not include preview audio
- Background playback is not fully supported
